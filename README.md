Library for basic ML in C++ with Python bindings

## Installation
To set up install install Pybind11:
```
python3 -m pip install pybind11[global]
```
and then run the following command:
```bash
python3 -m pip install basic_ml
```

## Usage

```python   
import basic_ml
```
